# NADIR

Aplicación Android de biofeedback respiratorio para la gestión de crisis de ansiedad.

![Captura principal](images/screenshot_main.png)

NADIR utiliza el acelerómetro del teléfono para detectar los movimientos respiratorios y proporcionar retroalimentación en tiempo real mediante canales visuales, hápticos y auditivos. El sistema ha sido diseñado para funcionar con los ojos cerrados reconociendo que durante un ataque de pánico la capacidad de atención visual se ve comprometida.

![Captura sesión](images/screenshot_session.png)

Todo el procesamiento se realiza localmente en el dispositivo sin transmitir información a servidores externos. Los protocolos respiratorios implementados incluyen la técnica 4-7-8 para relajación profunda, la respiración cuadrada para recuperación del enfoque y la respiración coherente para optimización de la variabilidad cardíaca.

![Captura resultados](images/screenshot_results.png)
